ESP32 binaries
===================
- application.bin
- bootloader.bin
- partitions.bin
- firmware.bin

application.bin + bootloader.bin + partitions.bin -> firmware.bin

Flash firmware.bin using the esptool or the run make deploy in the upy/ports/esp32.

- micropython.bin
